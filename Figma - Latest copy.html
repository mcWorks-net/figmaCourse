DESCRIPTION

Figma is a very powerful web-based application. While it shares similarities with Sketch and Adobe XD, it stands out as the most robust choice for team collaboration and rapid, responsive design. This crash course will delve into the Figma interface, helping you locate powerful tools that enable responsive design. Additionally, it will cover the creation of components to build a library of design elements for use throughout your projects. The course will guide you through the process of designing and prototyping, which includes working with artboards, crafting custom icons, editing images and text, and all the essential skills needed to create a polished and realistic web design.
